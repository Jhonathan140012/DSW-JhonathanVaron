  • Proposito del Sitio

    • La siguiente pagina tiene como proposito dar a conocer los temas que son: Flex Box, Listas y enlaces, donde se mostrar la ultidad que tienen las listas y Enlaces en el diseño web. Ademas de como funcionan los flexbox, ya que estos son una forma de organizar los elementos en una pagina web.

  • ¿Qué Flexbox?

    • Flexbox es un método de diseño de página unidimensional para compaginar elementos en filas o columnas. Los elementos de contenido se ensanchan para rellenar el espacio adicional y se encogen para caber en espacios más pequeños.

    Modelos Flexbox

  • Listas

    • Las listas en HTML ordenadas son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden

    Cada uno de los elementos de la lista ordenada se representará mediante el elemento LI.

    <ul>

      <li>Elemento 1</li>

      <li>Elemento 2</li>

    <ul>

  • Enlaces

    • Antes de crear enlaces en HTML debemos de tener presente que es una URL y cuáles son sus partes. URL son las siglas en inglés de Uniform Resource Locator, que en español significa Localizador Uniforme de Recursos.

    En HTML los enlaces se marcan con la etiqueta <a></a> y el atributo principal es href="/" donde se escribe la ubicación del archivo de destino que puede estar en la misma carpeta que el archivo que lo está llamando, en otra carpeta del mismo sitio o en otro sitio web.

    Ejemplo

    <a href="/contacto.html">Contáctenos</a>.
